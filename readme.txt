Adds address indexes to orders on WooCommerce that don't have them yet.

== Description ==

This plugins includes a new tool inside "WooCommerce" > "Status" > "Tools" to add address indexes to orders that don't have them yet.

Note that you should only run this plugin one time to create orders after upgrade from WooCommerce 2.6 or early, since orders created after WooCommerec 3.0 should already have properly indexes.

== Frequently Asked Questions ==

= When I can use this plugin? =

After update from WooCommerce 2.6 or early to WooCommerce 3.6.

= Where can I report bugs or contribute to this plugin? =

Join in on our [GitHub repository](https://github.com/woocommerce/woocommerce-recreate-orders-indexes/).
